Adding Images to Our HTML

The img Tag

The HTML tag used for adding images is a self-closing tag and has a few attributes that must be set. The first attribute is the src. The src attribute, pronounced source, is used to tell the browser where to find the image. The next attribute that is required for the element to be considered valid is the alt or alternate text attribute. The alt attribute is a short block of text telling the user what the image is. This is helpful information for someone using a screen reader or for the browser to still provide the information if the image doesn't display for whatever reason. A good resource for finding what properties are available to a given element is the W3Schools HTML Tag Reference - <img>.

Example

In the example below, we see an image that is properly displaying on the page.

<img src="../IMG/"> 
<html> 
	<head> 
		<title>Basic HTML Skeleton</title> 
		<meta charset="utf-8"> 
		<meta name="viewport" content="width=device-width, initial-scale=1.0"> 
		<link rel="stylesheet" href="styles.css"> <!-- The href value is the path to your stylesheet --> 
	</head> 
	<body> 
		<!-- Everything that is visible in the browser window should go here inside the body tag --> 
	</body> 
</html>

Javascript

Javascript is a client-side web programming language. The code that is written executes on the device that is viewing the web page. Javascript is an Object Based language and not Object Oriented. Similarly to using an external stylesheet, javascript is typically written in an external file. This allows functions and code to be reused throughout many pages on a website. It also helps separate the presentation of the pages from the logic and event handling that is being used to make our pages more dynamic. If something happens or needs to happen on our pages, we know to go look at the javascript and add the functionality there.
